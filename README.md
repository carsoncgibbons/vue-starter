## Installation
Install via the [Cosmic CLI](https://github.com/cosmicjs/cosmic-cli).
```bash
npm i -g cosmic-cli

# Login to your Cosmic JS account
cosmic login

# Installs example content to a new or existing Bucket and downloads the app locally
cosmic init vue-starter
cd vue-starter
cosmic start
```
